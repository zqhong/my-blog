# Scikit-learn Windows 安装
访问 http://www.lfd.uci.edu/~gohlke/pythonlibs 下载对应的 whl 文件。

numpy-1.10.4+mkl-cp27-cp27m-win32.whl
cp27： python2.7
win32： 32 位系统

安装
pip install /path/to/xxx.whl（绝对路径，不然可能会出错）

这里需要：
NumPy
Scipy
Scikit-learn
