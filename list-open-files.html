lsof - list open files

Posted by zqhong on Sun 16 October 2016

简单用法

# 查看所有网络连接
sudo lsof -i
# 仅所有 IPV6 连接
sudo lsof -i 6
# 仅显示 TCP 连接
sudo lsof -iTCP
# 使用-i:port来显示与指定端口相关的网络信息
sudo lsof -i:22
#使用@host来显示指定到指定主机的连接
sudo lsof -i@172.16.12.5

NOTE：某些信息需要有权限才能访问，必要时加上 sudo 或使用 root 用户执行 lsof 命令。
